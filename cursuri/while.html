while


Bucla while este, de asemenea, utilizată pentru a repeta numărul de instrucțiuni de mai multe ori. Cu toate acestea, dacă nu cunoaștem în avans numărul de iterații, se recomandă utilizarea unei bucle while. Spre deosebire de buclă, inițializarea și creșterea / descreșterea nu au loc în instrucțiunea buclă în bucla while.

Este, de asemenea, cunoscut sub numele de buclă controlată de intrare, deoarece starea este verificată la începutul buclei. Dacă condiția este adevărată, atunci corpul buclei va fi executat; în caz contrar, instrucțiunile după buclă vor fi executate.

Sintaxa:

while(condiție){
   //instructiuni care se vor repeta
}

Exemplu

Exemplu.java

public class Exemplu{
   public static void main(String args[]){
      int i = 0;
      System.out.println("Afisare primele pare mai mici decat 10:\n");
      while(i <= 10){
         System.out.println(i);
         i = i + 2;
      }
   }
}
